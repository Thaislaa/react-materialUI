# GrowAvalia - Exercício com React e Material UI

## 📌 Sobre o projeto

Este projeto foi desenvolvido como exercício prático utilizando **React** e **Material UI (MUI)**.

O objetivo foi reproduzir uma **tela de login** utilizando os componentes do MUI e implementar a **alternância de tema (Light/Dark)** com `ThemeProvider`.

---

## 🚀 Acesse o projeto online

🔗 **Deploy na Vercel:**
👉 [https://react-material-ui-beta.vercel.app/](https://react-material-ui-beta.vercel.app/)

---

## 🎯 Objetivos do exercício

* Reproduzir a interface proposta
* Utilizar componentes do Material UI
* Criar temas personalizados (light e dark)
* Alternar entre os temas utilizando `useState`
* Aplicar o tema globalmente com `ThemeProvider`

---

## 🖥️ Funcionalidades

* AppBar com título **GrowAvalia**
* Botão para alternar entre tema claro e escuro
* Formulário de login com:

  * Campo de E-mail
  * Campo de Senha
  * Botão de Login
* Aplicação de `CssBaseline` para padronização visual

---

## 🛠️ Tecnologias utilizadas

* React
* TypeScript
* Material UI (MUI)
* Emotion (`ThemeProvider`)
* Vercel
